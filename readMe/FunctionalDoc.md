###시스템 구현

---
- 사용 기술
  - React, Axios, JQuery, Spring Boot 2.4.2, Spring Security 
- 사용 도구
   - 소프트웨어 개발 도구 : IntelliJ
   - 데이터베이스: MySQL 8.0.23
   - WAS : Tomcat 9.0.41
   - 배포 도구 : Gradle
   - 형상관리 도구 : Github
